Honours Dissertation 2025 – 2026
Adversarial AI Competition Platform
Overview

An end-to-end web platform that automates the evaluation of adversarial machine learning attack and defense algorithms. Built as a final-year Honours dissertation under Dr. Xiaowei Huang, the platform replaces a manual marking process for COMP329 Advanced AI students.

Key Features
  • Students upload Python attack scripts and PyTorch (.pt) defense models via a secure web interface
  • Sandboxed Docker containers execute each submission against 10 baseline attacks and 10 baseline defenses
  • Automated scoring engine returns accuracy metrics and adversarial robustness scores instantly
  • Live leaderboard ranks submissions across the cohort in real-time
  • JWT-secured authentication with role-based access for students vs. administrators
  • PostgreSQL database stores submissions, scores, and evaluation history with full SQL querying
Architecture

React frontend → Flask REST API backend → Docker evaluation pipeline → PostgreSQL database. Supervisor: Dr. Xiaowei Huang. Based on COMP329 coursework from the AISafetyLectureNotes textbook.

COMP338 Coursework 2025
Computer Vision: Feature Extraction & Edge Detection
Overview

A deep technical investigation into low-level computer vision algorithms. Implemented Canny edge detection from scratch in Python, then conducted a rigorous comparative analysis of three major feature descriptor algorithms across varied image datasets.

Key Features
  • Full Canny edge detection pipeline implemented from scratch — Gaussian smoothing, gradient calculation, non-maximum suppression, hysteresis thresholding
  • Comparative benchmark of SIFT, SURF, and ORB feature descriptors across multiple image categories
  • Performance analysis under varying lighting, scale, rotation, and noise conditions
  • Visualisation suite built with Matplotlib showing keypoint distributions and match quality
  • Statistical evaluation metrics including precision, recall, and match repeatability
